## Inspection Results

### Stack Verification

- **Language**: Python 3.11 (Docker: `python:3.11-slim`)
- **Framework**: FastAPI + uvicorn (`app/main.py`)
- **Face engine**: DeepFace, model `Facenet512` (512-dim), detector `retinaface` (configurable via `.env`)
- **Vector DB**: PostgreSQL 16 + pgvector (`pgvector/pgvector:pg16` image) with HNSW index on cosine distance
- **Image storage**: DigitalOcean Spaces (S3-compatible, `boto3`)
- **DB driver**: psycopg (v3) + psycopg_pool.ConnectionPool
- **Config**: pydantic-settings (`.env` file)
- **Schemas**: pydantic v2 (`app/schemas.py`)
- **Deployment**: docker-compose (api + db services)
- **Process manager**: pm2 on the droplet (service: `rostros-api`)
